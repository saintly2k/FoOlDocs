Customizing your Installation

It is recommended to Copy the Folder "RoriconScans" and make the changes there, but you can do it how you want to.

Simply copy the Folder and rename it to what you want.

You can edit these files:

  • edit.css
  • login.css
  • main.css
  • pagination.css

You can't edit these file (or else you will break the hero-slider):

  • w3.css
